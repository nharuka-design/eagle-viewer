Eagle Viewer

Google Drive の Eagle ライブラリを iPhone・iPad・PC で閲覧できます。

Google Drive フォルダリンク必須
Eagle の .library フォルダを右クリック → 共有 → 「リンクを知っている全員・閲覧者」に設定してリンクをコピー
Google API キー必須
APIキーの取得手順を表示 ▾
1
console.cloud.google.com にアクセス・ログイン
2
新しいプロジェクトを作成（名前は何でも可）
3
「APIとサービス」→「ライブラリ」→ Google Drive API を検索して有効化
4
「認証情報」→「認証情報を作成」→「APIキー」→ 表示されたキーをコピー
5
キーの編集 → 「HTTPリファラー制限」→ https://あなたのGitHubユーザー名.github.io/* を追加（推奨）
