Eagle Viewer

Google Drive の Eagle ライブラリを iPhone・iPad・PC で閲覧できます。

Google Drive フォルダリンク 必須
Eagle の .library フォルダを右クリック → 共有 → 「リンクを知っている全員」に設定してリンクをコピー
Google API キー 必須
Google Cloud Console で無料取得 → 取得手順を表示 ▾
1
Google Cloud Console（console.cloud.google.com）にアクセス・ログイン
2
上部の「プロジェクトを選択」→「新しいプロジェクト」→ 任意の名前で作成
3
左メニュー「APIとサービス」→「ライブラリ」→ 「Google Drive API」を検索 → 有効にする
4
「認証情報」→「認証情報を作成」→「APIキー」→ 表示されたキーをコピー
5
（推奨）作成したAPIキーの「編集」→「アプリケーションの制限」→「HTTPリファラー」→ https://あなたのGitHub名.github.io/* を追加すると他人が使えなくなり安全
